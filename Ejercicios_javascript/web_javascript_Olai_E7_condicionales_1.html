Ejercicio 07 — Introducción de datos y toma de decisiones

Este programa pregunta al usuario si quiere participar. Si responde que sí, le pide que introduzca una fruta y, mediante un switch, muestra su precio si está disponible. Después realiza una segunda pregunta para saber si quiere consultar algo más. Dependiendo de las respuestas, el programa toma diferentes decisiones. Si el usuario no quiere participar o introduce una respuesta incorrecta, el programa muestra un mensaje adecuado.
